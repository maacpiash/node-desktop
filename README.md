# Template for Node.js desktop application
> with Electron, React, TypeScript and Webpack

## Install

Installation of packages (Yarn is preferred):

```sh
yarn
```

## Usage

### To build the application: 

```sh
yarn build
```

### To start the application:

```sh
yarn start
```
